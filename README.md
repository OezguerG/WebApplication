# General  
Dieses Projekt entstand im Rahmen der Vorlesung *Webengineering 2*.  

Es handelt sich um die Entwicklung einer Full-Stack-Anwendung mit folgenden Kernfunktionen:  
- Benutzer-Authentifizierung  
- REST-API  
- Responsives UI  

**Technologien:**  
- Frontend: React  
- Backend: Node.js  
- Datenbank: MongoDB  

---
